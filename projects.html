VeriTrade Mobile App

The system leverages intelligent data extraction and machine learning techniques to analyze invoices, validate supplier information, and generate risk-based insights, enabling businesses to make safer financial decisions.

Key Features

  • Invoice Upload & Processing: Users can upload invoices in PDF or image format
  • Automated Data Extraction: Extracts key invoice details such as supplier name, amount, and dates
  • Supplier Verification: Cross-checks supplier information against public business registration databases
  • AI Fraud Detection: Detects suspicious patterns using machine learning models
  • Risk Scoring System: Generates a trust score and actionable recommendations for each supplier
  • Cross-Platform Access: Accessible via both web and mobile applications
  • Real-Time Feedback: Provides instant analysis results to support decision-making

Tech Stack

  • Frontend
    React Native (Expo) for Mobile application
    React.js for Web interface
  • Backend
    Node.js
    Express.js for API development
  • Database
    MySQL for Structured data storage
  • DevOps & Deployment
    Docker for Containerization of backend services
    GitHub for Version control and collaboration
    GitHub Actions for CI/CD pipelines
    Microsoft Azure for Cloud hosting and deployment
  • AI and Data Processing
    OCR tools for invoice data extraction
    Machine Learning models for fraud detection and risk scoring

This project was developed as part of a collaborative team effort under the Women Techsters Fellowship program.
My Role as the DevOps Engineer:

  • Designed and implemented CI/CD pipelines using GitHub Actions
  • Containerized backend services using Docker
  • Managed environment configurations (development, staging, production)
  • Deployed and maintained backend infrastructure on Microsoft Azure
  • Collaborated with cross-functional teams to ensure smooth integration and delivery
